Capitolo 3: HTML Semantico e Accessibilità

L'HTML non è solo un linguaggio di markup, è un linguaggio semantico che comunica il significato del contenuto, non solo la sua apparenza.

Perché la Semantica Conta

HTML semantico ha benefici concreti:

  • Accessibilità: screen reader capiscono la struttura
  • SEO: motori di ricerca valorizzano markup significativo
  • Manutenibilità: codice auto-documentante
  • Styling: CSS può targettizzare elementi semantici

Tag Strutturali nell'Esercizio

<!-- Struttura semantica -->
<header>
  <h1>Titolo principale</h1>
  <nav>...</nav>
</header>

<main>
  <article>...</article>
</main>

<footer>...</footer>

Ogni tag comunica ruolo e scopo:

  • <header> → introduzione/intestazione
  • <nav> → navigazione principale
  • <main> → contenuto principale (unico per pagina)
  • <article> → contenuto autocontenuto
  • <footer> → informazioni di chiusura

Heading Hierarchy

Gli heading (h1-h6) creano un outline del documento: 

h1 - Titolo capitolo (uno per pagina)
  h2 - Sezione principale
    h3 - Sottosezione
      h4 - Dettaglio specifico

Regola d'oro: Mai saltare livelli. Non passare da h2 a h4 saltando h3.

ARIA e Accessibilità

ARIA (Accessible Rich Internet Applications) migliora l'accessibilità: 

<button aria-label="Esegui codice">▶</button>

<div role="alert">Errore nel codice</div>

Best Practices ARIA

  • Usa HTML semantico prima di ARIA
  • Non ridefinire semantica nativa
  • Testa con screen reader reali

Contrasto e Leggibilità

WCAG 2.1 richiede rapporti di contrasto minimi:

  • Testo normale: almeno 4.5:1
  • Testo grande (18pt+): almeno 3:1

Nell'Esercizio: #2a2a2a su #fafafa = 13.8:1 ✓

Focus Management

La navigazione da tastiera deve essere ovvia: 

/* Focus visibile */
a:focus, button:focus {
  outline: 2px solid var(--accent-green);
  outline-offset: 2px;
}

Riepilogo

  • ✓ HTML semantico per significato, non solo forma
  • ✓ Heading hierarchy logica
  • ✓ ARIA quando HTML non basta
  • ✓ Contrasto adeguato per leggibilità
  • ✓ Focus management per navigazione da tastiera
