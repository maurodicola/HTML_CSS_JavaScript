Capitolo 4: Sistema di Design CSS

Un design system non è solo una raccolta di colori e font. È un linguaggio visivo coerente che guida tutte le decisioni estetiche.

CSS Custom Properties (Variabili)

Il cuore del design system dell'Esercizio: 

:root {
  /* Palette cromatica */
  --bg-main: #fafafa;
  --text-primary: #2a2a2a;
  --accent-yellow: #f4e8c1;
  --accent-green: #d4e7d0;
  
  /* Spaziature */
  --spacing-sm: 1rem;
  --spacing-md: 2rem;
  --spacing-lg: 3rem;
  
  /* Tipografia */
  --font-display: 'Crimson Pro', serif;
  --font-body: 'DM Sans', sans-serif;
}

Vantaggi delle Variabili CSS

  • Cambio globale con un edit
  • Consistenza garantita
  • Manutenibilità migliorata
  • Theming facile (es. dark mode)

Scala Tipografica

La tipografia segue una progressione matematica:

Elemento Dimensione Rapporto
h1 3rem (48px) Base × 3
h2 2rem (32px) Base × 2
h3 1.5rem (24px) Base × 1.5
body 1rem (16px) Base

Questa scala crea armonia visiva e gerarchia chiara.

Palette Cromatica Minimalista

Scelte cromatiche dell'Esercizio:

  • Grigio neutro (#fafafa): sfondo non aggressivo
  • Grigio scuro (#2a2a2a): testo leggibile
  • Giallo pastello: calore, positività
  • Verde pastello: calma, natura
"Il colore non è decorazione, è comunicazione."

Sistema di Spaziature

Spaziature consistenti creano ritmo visivo: 

--spacing-xs: 0.5rem  (8px)
--spacing-sm: 1rem    (16px)
--spacing-md: 2rem    (32px)
--spacing-lg: 3rem    (48px)
--spacing-xl: 4rem    (64px)

Multipli di 8px facilitano allineamenti pixel-perfect.

Responsive Typography

Font sizes si adattano al viewport: 

/* Base: 16px */
html { font-size: 16px; }

/* Mobile: 14px */
@media (max-width: 768px) {
  html { font-size: 14px; }
}

Usando rem, tutto scala proporzionalmente.

Componenti Riusabili

Ogni pattern visivo è codificato come classe:

  • .intro-box → box giallo introduttivo
  • .tip-box → box verde suggerimenti
  • .warning-box → box grigio avvisi
  • .btn → pulsanti standard

Componi pagine combinando questi building blocks.

Performance CSS

Ottimizzazioni implementate:

  • Selettori semplici (evita nesting profondo)
  • Nessuna ridondanza (DRY principle)
  • CSS minimizzato per produzione
  • Font loading ottimizzato con preconnect

Riepilogo

  • ✓ Palette cromatica intenzionale
  • ✓ Sistema di spaziature consistente
  • ✓ Componenti riusabili
  • ✓ Performance-first approach
